Recovering
==========

| Variable           | Value             |
| ------------------ | ----------------- |
| `$XDG_CONFIG_HOME` | ~/.config         |
| `$XDG_DATA_HOME`   | ~/.local/share    |

## Symlinks

 - ~/.bashrc -> ./bashrc
 - ~/.config/fontconfig -> ./fontconfig
 - ~/.vim -> ./vim
 - ~/.vimrc -> ./vimrc
 - ~/.theme/Packed Ice Theme -> [satgo1546/packed-ice-theme][1]

## Copy these

 - C:\Windows\Fonts -> ~/.local/share/fonts
